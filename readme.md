# Chatbot Setup

add assets/configs/commands.json

example:
```json {
  "commands": [
    {
      "name": "!dc",
      "response": "<your Discord link>"
    },
    {
      "name": "!hello",
      "response": "Hi"
    }
  ]
}
```
